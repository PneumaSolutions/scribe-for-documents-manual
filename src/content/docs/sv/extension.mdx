Välkommen till webbläsartillägget Scribe for Documents! Det här tillägget gör det enklare än någonsin att konvertera och få tillgång till dina dokument i ett tillgängligt format, oavsett var du befinner dig. Med bara några klick kan du ladda upp filer, konvertera dem direkt från din webbläsare och få de resultat du behöver i ett format som fungerar för dig. 

## Så här använder du webbläsartillägget Scribe for Documents:

<Windows>
* Ladda upp filer direkt: Klicka på tilläggsikonen eller tryck på *Alt+Shift+O* för att öppna filuppladdningsfönstret och välj det dokument som du vill ska åtgärdas direkt.
    * Observera att det här kortkommandot kan ändras via webbläsarens möjlighet att anpassa kortkommandon.
* Konvertering med högerklick: Hittade du en dokumentlänk online? Högerklicka på den eller tryck på *Shift+F10* och välj "Konvertera med Scribe" för att behandla den på plats.
* Automatisk konverteringsprompt: När du laddar ner en fil som Scribe stöder får du ett systemmeddelande som frågar om du vill konvertera den.
</Windows>

<MacOS>
* Ladda upp filer direkt: Klicka på tilläggsikonen eller tryck på *Command+Shift+O* för att öppna fönstret för filuppladdning och välj det dokument som du vill ska åtgärdas direkt.
    * Observera att den här genvägen kan ändras via webbläsarens möjlighet att anpassa tangentbordsgenvägar.
* Konvertering med högerklick: Hittade du en dokumentlänk online? Högerklicka på den eller tryck på *CTRL+Option+Shift+M* och välj "Konvertera med Scribe" för att bearbeta den på plats.
* Automatisk konverteringsprompt: När du laddar ner en fil som stöds av Scribe får du ett systemmeddelande som frågar om du vill konvertera den.
</MacOS>

Detta tillägg är utformat för att effektivisera dokumenttillgängligheten och samtidigt säkerställa en sömlös användarupplevelse. Om du har några frågor kan du kolla in våra supportresurser eller [kontakta vårt supportteam](https://pneumasolutions.com/contact) för att få hjälp.
