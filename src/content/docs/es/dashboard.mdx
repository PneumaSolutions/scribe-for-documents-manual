Scribe cuenta con un panel de configuración para facilitar diversos ajustes de corrección de documentos.

### Valores predeterminados del documento 

La página Valores predeterminados del documento, situada en el menú Configuración, permite establecer valores predeterminados globales que se mantienen en todas las correcciones. Entre ellos se incluyen:
* Idioma
* Preferencias de descripción automática de imágenes
* Preferencias de texto a voz
    * Voz predeterminada
    * Velocidad predeterminada
* Tabla de traducción Braille
