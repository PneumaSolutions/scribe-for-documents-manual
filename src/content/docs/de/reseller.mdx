Wenn Ihre Organisation als Wiederverkäufer eingerichtet ist, sehen Sie ein Wiederverkäufer-Dashboard im Navigationsmenü der Landing Page Ihres Kontos. Dieses Dashboard ermöglicht Ihnen die Verwaltung der untergeordneten Organisationen Ihrer Organisation.  
Sobald Sie sich auf dem Dashboard befinden, erhalten Sie einen Überblick darüber, wie viele Credits sich auf Ihrem Hauptkonto befinden und wie viele übertragen wurden.

### Hinzufügen einer Organisation

Bevor Sie Credits übertragen können, müssen Sie eine untergeordnete Organisation hinzufügen. Klicken Sie auf den Link "Organisation hinzufügen", um den Vorgang zu starten.  
Wenn Sie eine Organisation hinzufügen, müssen Sie deren Namen, die Domäne und die E-Mail-Adresse der Hauptkontaktperson angeben. Optional können Sie den Vor- und Nachnamen des Hauptansprechpartners hinzufügen. Nachdem Sie die Sprache der Organisation festgelegt haben, klicken Sie auf "Speichern".  
Beachten Sie, dass Scribe for Documents nicht automatisch eine E-Mail an die primäre Kontaktperson sendet. Das System geht davon aus, dass Sie als Hauptkontoinhaber die Kommunikation mit der primären Kontaktperson der Organisation, an die Sie weiterverkaufen, übernehmen. Daher müssen Sie diese Person bezüglich ihres neuen Kontos kontaktieren.

### Organisationen verwalten

Sobald Sie mindestens eine untergeordnete Organisation hinzugefügt haben, sehen Sie einen Abschnitt "Kundenorganisationen" mit einer Liste der Organisationen unter Ihrem Dach. Klicken Sie auf eine Organisation, um sie zu verwalten.   

#### Kredite verwalten

Sie können der Organisation Seitenguthaben hinzufügen oder entfernen, indem Sie auf die entsprechenden Links klicken. Beim Hinzufügen oder Entfernen von Seitenguthaben werden Sie aufgefordert, den gewünschten Betrag über die Drehsteuerung einzugeben. Sobald der Vorgang abgeschlossen ist, wird Ihnen angezeigt, wie viele Credits die untergeordnete Organisation jetzt hat und wie viele Credits auf Ihrem Hauptkonto verbleiben.  

#### Versuche

Wenn Sie auf den Link "Testversion starten" klicken, können Sie eine Testphase für Ihre untergeordnete Organisation einrichten. Wenn Sie einen Testzeitraum gewähren, werden Sie nach der Anzahl der Seiten gefragt, die Sie auf das Konto der Organisation einzahlen möchten, sowie nach der Länge des Testzeitraums. Stellen Sie die Werte nach Ihren Wünschen ein und klicken Sie dann auf die Schaltfläche "Seiten hinzufügen", um den Aktivierungsprozess abzuschließen.  
Eine Testversion kann bis zu 30 Tage dauern, entweder anfänglich oder durch eine Verlängerung. Sie können bis zu 200 Seiten kostenlos hinterlegen, d. h. sie werden nicht vom Guthaben Ihres Kontos abgezogen. Alle Seiten, die diese Grenze überschreiten, werden von Ihrem Guthaben abgezogen, aber alle ungenutzten Seiten werden nach Ablauf des Testzeitraums wieder auf Ihr Konto eingezahlt. Nehmen wir an, Sie gewähren einer Kinderorganisation eine 15-tägige Testphase mit 500 Seiten. Die ersten 200 Seiten sind kostenlos, und die restlichen 300 werden von Ihrem Konto abgezogen. Wenn die Kinderorganisation 300 Seiten nutzt und nach Ablauf der Testphase noch 200 Seiten übrig sind, werden die restlichen 200 Seiten der 300 abgehobenen Guthaben auf Ihr Konto zurücküberwiesen.  
Wenn Sie die Testphase einer Organisation verlängern möchten, können Sie dies über den Link "Testphase verlängern" tun. Sie können die Testphase jedoch nur bis zu einer Gesamtdauer von 30 Tagen verlängern.

### Zeitrahmen für Seitenguthaben

Die Verfallsdaten der Seitenguthaben für Ihr Konto und die Ihrer untergeordneten Organisationen sind in Ihrem Wiederverkäufervertrag festgelegt.  
Bei jeder Testversion, deren Guthaben 200 übersteigt und die somit Guthaben von Ihrem Konto abzieht, wird das ungenutzte Guthaben nach Ablauf der Frist an das Hauptkonto Ihrer Organisation zurückerstattet.
