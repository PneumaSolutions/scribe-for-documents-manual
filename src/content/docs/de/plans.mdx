Scribe bietet eine Vielzahl von Plänen und On-Demand-Kaufoptionen, die Ihren Bedürfnissen entsprechen.

Der Link [Erfahren Sie, wie Sie mit den Scribe Augmented Document Remediation-Paketen Geld sparen können](https://scribeit.io/signup/documents_upsell) auf Ihrem Konto-Dashboard führt Sie zu einer Seite, auf der Sie einen Plan oder ein vorausbezahltes Seitenpaket kaufen können. Sie können auch [Guthaben auf das Scribe-Konto einer anderen Person aufladen.](https://scribeit.io/signup/account_credit?state=%7B%22return_to%22%3A%22%2F%22%7D)

### On-Demand Behebung

Bei der On-Demand-Remediation können Sie eine einzelne Konvertierung kaufen. Der Preis richtet sich nach der Anzahl der Seiten des Dokuments.

Beachten Sie, dass diese Option nur angezeigt wird, wenn Sie eine Bestellung direkt aus dem Bereich "Vollständiges Dokument bereinigen" aufgeben.

### Prepaid Page Pack

Mit einem vorausbezahlten Seitenpaket erhalten Sie 100 Seiten zu einem deutlich günstigeren Preis als bei der Bezahlung pro Seite. Diese Seitenguthaben verfallen nicht.

### Jahresabonnement

Ein Jahresabonnement bietet 2500 Seitenguthaben sowie einen Rabatt von 30 % auf die Human-Remediation. Diese Credits können nicht verlängert werden.
