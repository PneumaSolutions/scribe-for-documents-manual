Willkommen bei der Browser-Erweiterung Scribe for Documents! Diese Erweiterung macht es einfacher denn je, Ihre Dokumente in ein zugängliches Format zu konvertieren und darauf zuzugreifen, egal wo Sie sind. Mit nur wenigen Klicks können Sie Dateien hochladen, sie direkt von Ihrem Browser aus konvertieren und die gewünschten Ergebnisse in einem für Sie geeigneten Format erhalten. 

## Wie man die Scribe for Documents Browser-Erweiterung verwendet:

<Windows>
* Direktes Hochladen von Dateien: Klicken Sie auf das Erweiterungssymbol oder drücken Sie *Alt+Umschalt+O*, um das Datei-Upload-Fenster zu öffnen und wählen Sie das Dokument aus, das Sie sofort korrigieren möchten.
    * Beachten Sie, dass diese Tastenkombination über die Funktion zur Anpassung der Tastaturkürzel Ihres Browsers geändert werden kann.
* Rechtsklick-Konvertierung: Sie haben online einen Link zu einem Dokument gefunden? Klicken Sie mit der rechten Maustaste darauf oder drücken Sie *Umschalt+F10* und wählen Sie "Mit Scribe konvertieren", um das Dokument sofort zu bearbeiten.
* Automatische Konvertierungsaufforderung: Wenn Sie eine von Scribe unterstützte Datei herunterladen, erhalten Sie eine Systembenachrichtigung mit der Frage, ob Sie die Datei konvertieren möchten.
</Windows>

<MacOS>
* Direktes Hochladen von Dateien: Klicken Sie auf das Erweiterungssymbol oder drücken Sie die Tastenkombination *Befehl+Umschalt+O*, um das Datei-Upload-Fenster zu öffnen und das Dokument auszuwählen, das Sie sofort konvertieren möchten.
    * Beachten Sie, dass diese Tastenkombination über die Funktion zur Anpassung der Tastaturkürzel Ihres Browsers geändert werden kann.
* Rechtsklick-Konvertierung: Sie haben online einen Link zu einem Dokument gefunden? Klicken Sie mit der rechten Maustaste darauf oder drücken Sie *Strg+Wahl+Umschalt+M* und wählen Sie "Mit Scribe konvertieren", um das Dokument auf der Stelle zu bearbeiten.
* Automatische Konvertierungsaufforderung: Wenn Sie eine Datei herunterladen, die von Scribe unterstützt wird, erhalten Sie eine Systembenachrichtigung mit der Frage, ob Sie die Datei konvertieren möchten.
</MacOS>

Diese Erweiterung wurde entwickelt, um die Zugänglichkeit von Dokumenten zu optimieren und gleichzeitig eine nahtlose Benutzererfahrung zu gewährleisten.
