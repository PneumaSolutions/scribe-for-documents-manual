Scribe  features a  settings panel to facilitate various document  remediation adjustments.

### Document Defaults 

The Document Defaults page, located under the Settings menu, allows you to set global defaults that persist across all remediations. They include:
* Language
* Automatic image description preference
* Text to speech preferences
    * Default voice
    * Default speed
* Braille translation table
