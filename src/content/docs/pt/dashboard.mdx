O Scribe apresenta um painel de configurações para facilitar vários ajustes de correção de documentos.

### Padrões de documentos 

A página Document Defaults (Padrões do documento), localizada no menu Settings (Configurações), permite que você defina padrões globais que persistem em todas as correções. Elas incluem:
* Idioma
* Preferência de descrição automática de imagem
* Preferências de texto para fala
    * Voz padrão
    * Velocidade padrão
* Tabela de tradução em Braille
