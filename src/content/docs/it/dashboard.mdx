Scribe dispone di un pannello di impostazioni per facilitare le varie regolazioni della correzione dei documenti.

### Predefiniti del documento 

La pagina delle impostazioni predefinite del documento, situata nel menu Impostazioni, consente di impostare le impostazioni predefinite globali che persistono in tutte le riparazioni. Esse includono:
* Lingua
* Preferenze per la descrizione automatica delle immagini
* Preferenze per la sintesi vocale
