Benvenuti nell'estensione del browser Scribe for Documents! Questa estensione rende più facile che mai convertire e accedere ai vostri documenti in un formato accessibile, ovunque vi troviate. Con pochi clic, è possibile caricare i file, convertirli direttamente dal browser e ottenere i risultati desiderati in un formato adatto alle proprie esigenze. 

## Come utilizzare l'estensione del browser Scribe for Documents:

<Windows>
* Caricare direttamente i file: Fare clic sull'icona dell'estensione o premere *Alt+Shift+O* per aprire la finestra di caricamento dei file e scegliere il documento che si desidera rimediare istantaneamente.
    * Questa scorciatoia può essere modificata tramite la funzione di personalizzazione delle scorciatoie da tastiera del browser.
* Conversione con il tasto destro del mouse: Avete trovato un link a un documento online? Fate clic con il tasto destro del mouse o premete *Maiusc+F10* e selezionate "Converti con Scribe" per elaborarlo immediatamente.
* Prompt di conversione automatica: Quando si scarica un file supportato da Scribe, viene visualizzata una notifica di sistema che chiede se si desidera convertirlo.
</Windows>

<MacOS>
* Caricare direttamente i file: Fare clic sull'icona dell'estensione o premere *Command+Shift+O* per aprire la finestra di caricamento dei file e scegliere il documento che si desidera rimediare istantaneamente.
    * Questa scorciatoia può essere modificata tramite la funzione di personalizzazione delle scorciatoie da tastiera del browser.
* Conversione con il tasto destro del mouse: Avete trovato un link a un documento online? Fate clic con il tasto destro del mouse o premete *CTRL+Option+Shift+M* e selezionate "Converti con Scribe" per elaborarlo immediatamente.
* Prompt di conversione automatica: Quando si scarica un file supportato da Scribe, viene visualizzata una notifica di sistema che chiede se si desidera convertirlo.
</MacOS>

Questa estensione è stata progettata per semplificare l'accessibilità ai documenti e garantire un'esperienza utente senza soluzione di continuità.
